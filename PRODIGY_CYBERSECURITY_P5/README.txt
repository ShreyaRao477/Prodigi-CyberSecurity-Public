Packet Sniffer Tool

This packet sniffer tool is intended for educational and ethical purposes only.

Before using this tool, please ensure that you:
1. Have explicit permission to monitor and capture network traffic on the network.
2. Use this tool responsibly and for ethical purposes only.

Key Features:
- Captures packets based on TCP, UDP, or both protocols.
- Displays packet details including source IP, source port, protocol, and a portion of the payload.
- Saves captured packets' details in a text file.
- Provides graphical visualizations of captured data (Source IPs, Source Ports, and Protocols).
- Allows users to start a new sniffing session or view previously captured results.

How to Use:
1. Run the script.
2. Accept the terms and conditions when prompted.
3. Choose the protocol (TCP, UDP, or Both) you wish to capture.
4. Enter the duration for which you want the sniffing to take place.
5. The sniffing will start, and the captured packets will be displayed and saved to 'packet_sniffer_results.txt'.
6. After the sniffing session, you will have the option to:
   - View the graphical data of captured packets.
   - View the text file data.
   - Start a new sniffing session.
   - Quit the tool.

Requirements:
- Python 3.x
- Scapy (install via pip)
- Matplotlib (install via pip)
