------------------ Keylogger Program README ------------------

This keylogger program is intended for educational and ethical purposes only. It allows you to capture keystrokes for a specified duration and store them in a log file. This tool is designed to be used on systems where you have explicit permission to monitor and log keystrokes.

---------------------------------- Features ----------------------------------
1. **Log Keystrokes**: Capture keystrokes for a specified duration and save them in a log file.
2. **View Log**: View the log file or display its contents directly in the terminal.
3. **Keylogger Issues and Concerns**: Learn about the ethical, legal, and security issues related to the use of keyloggers.

---------------------------- Program Usage -----------------------------------
1. **Disclaimer Acceptance**:
   - Upon running the program, you must accept the terms and conditions to proceed.
   
2. **Logging Keystrokes**:
   - Enter the duration in seconds for which the keystrokes should be logged.
   - The program will record all keystrokes during this period and save them to a file named `keylogger_log.txt`.

3. **Menu Options**:
   After logging the keystrokes, you will be presented with a menu of options:
   - **Option 1**: View the path of the saved log file.
   - **Option 2**: Display the contents of the log file in the terminal.
   - **Option 3**: Learn about the ethical, legal, and security issues associated with keyloggers.
   - **Option 4**: Quit the program.

---------------------------------- File Locations --------------------------
- **Log File**: The log file where the captured keystrokes are saved is `keylogger_log.txt` and will be stored in the same directory as the script.
