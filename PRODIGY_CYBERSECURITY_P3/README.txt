Password Strength Checking Tool
-------------------------------
This tool checks the strength of a password based on a set of criteria and provides suggestions for strengthening it. It evaluates the password against the following factors:

1. Length: The password must be at least 8 characters long.
2. Numbers: The password must contain at least one numeric digit.
3. Uppercase: The password must contain at least one uppercase letter.
4. Lowercase: The password must contain at least one lowercase letter.
5. Special Characters: The password must contain at least one special character (e.g., !, @, #, $, etc.).

The program offers the following options:
1. **Check Password Strength**: Displays the strength level of the password based on the criteria met.
2. **View Suggestions for Strengthening Password**: Provides suggestions on how to improve the password strength.
3. **View Password Complexity**: Displays how many of the criteria the password meets (out of 5).
4. **Quit**: Exits the program.

Features:
---------
- The program checks the password against multiple criteria to assess its strength.
- It gives feedback on the current password strength level (Very Strong, Strong, Moderate, Weak).
- Based on the evaluation, the program provides personalized suggestions to enhance password security.
- The program includes a menu for users to select options for checking strength, viewing suggestions, or examining complexity.

How to Run:
------------
1. Run the Python script.
2. Enter a password when prompted.
3. Choose from the available options (1, 2, 3, or 4) based on your preference.
